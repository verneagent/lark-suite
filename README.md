# lark-wiki

`lark-wiki` is a public agent skill authored by Verne for reading and editing Lark wiki documents through the Open API.

## Install

```bash
npx skills add verneagent/lark-wiki
```

## Scope

- Supported: Claude Code, OpenCode

## Repository Layout

- `SKILL.md` is the skill entrypoint.
- `scripts/` contains the helper scripts.
